# Student Management System

## Description
This project is a web-based system that manages student accounts, roles, and approvals.

## Features
- User registration (student, cashier, finance)
- Admin approval system
- Login and authentication
- Role-based access

## Installation
1. Download test.zip
2. Import the database name user in the test in phpMyAdmin
3. Configure db.php
4. Run on localhost (XAMPP)

## Usage
- Register as different roles
- Wait for admin approval
- Login and access dashboard
